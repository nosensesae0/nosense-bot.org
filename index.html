Como Instalar (É Moleza!) 🛠️

No Termux (Android)

Cola esse comando no seu Termux e deixa rolar:

pkg upgrade -y && pkg update -y && pkg install git -y && pkg install nodejs-lts -y && pkg install ffmpeg -y && pkg install unzip -y && pkg install python python-pip && npm install moment jsdom axios cheerio && npm install node-summarizer && npm install google-translate-api-x && npm install natural stopword lodash compromise sentiment && npm install node-fetch whois-json qrcode libphonenumber-js google-libphonenumber wikijs mathjs

No Ubuntu/Linux

Para quem usa Linux, é assim:

sudo apt update && sudo apt upgrade -y
sudo apt install -y git nodejs ffmpeg unzip python3 python3-pip
npm install moment jsdom axios cheerio node-summarizer google-translate-api-x natural stopword lodash compromise sentiment node-fetch whois-json qrcode libphonenumber-js google-libphonenumber wikijs mathjs

Baixar e Rodar

Agora é só clonar o repositório e mandar ver:

wget https://github.com/bragadev1234/NoSenseBot/archive/refs/heads/main.zip -O nosense-bot.zip
unzip nosense-bot.zip
mv nosense-bot-Bot-main bot
cd bot
chmod -R 755 ./*
npm start
